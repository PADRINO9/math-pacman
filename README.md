# Math Maze — מבוך הכפל

משחק עצמאי ונפרד מהאתר הראשי. צד הלקוח אינו תלוי בחבילות חיצוניות או ב־build system.

הדמות הראשית היא Bifly. ספרייט הבסיס ופריימי האכילה נמצאים בקובצי `assets/bifly-*.png`.
ספרייט הבסיס והבעות הפנים של האויב נמצאים בקובצי `assets/dark-enemy*.png`.
פוסטר מסך הפתיחה נמצא ב־`assets/math-maze-poster.png`.

פותחים את `index.html` בדפדפן או מריצים שרת מקומי מתוך התיקייה:

```bash
python3 -m http.server 5178
```

ואז נכנסים לכתובת:

```text
http://localhost:5178
```

השרת הפשוט מציג את המשחק, אך אינו מריץ את פונקציית `/api/champions`. טבלת השיאים פועלת בפריסת Vercel או דרך `vercel dev`.

## אלוף האלופים

טבלת השיאים משתמשת ב־Supabase Free ובפונקציית Vercel שנמצאת ב־`api/champions.js`.
השחקן יכול לפרסם שיא רק לאחר 25 תשובות נכונות, כלומר לאחר מעבר השלב הראשון.

1. יוצרים פרויקט בתוכנית Free ב־[Supabase](https://supabase.com/).
2. פותחים את SQL Editor ומריצים את הקובץ `supabase/schema.sql`.
3. מגדירים בפרויקט Vercel את משתני הסביבה הבאים:

```text
SUPABASE_URL=https://YOUR_PROJECT.supabase.co
SUPABASE_SERVICE_ROLE_KEY=YOUR_SERVICE_ROLE_KEY
```

4. מבצעים פריסה חדשה כדי שמשתני הסביבה ייכנסו לתוקף.

אסור להכניס את `SUPABASE_SERVICE_ROLE_KEY` לקוד צד הלקוח או ל־Git. ה־API שומר אותו בצד השרת בלבד.

## מה שופר

- מבוך קבוע ויציב במקום מחסומים אקראיים שעלולים לחסום את המשחק.
- תנועה מבוססת זמן, התנגשויות מעוגלות ותור כיוון בתוך המבוך.
- יריבים עם בחירת מסלול, מצבי רדיפה/פיזור וצבעים מובחנים.
- מסך פתיחה שמבקש את שם השחקן לכל משחק ומציג אותו במסך הניצחון.
- מסך פתיחה אחוד ורספונסיבי שמשלב את הפוסטר בתוך הממשק במחשב ובמובייל.
- טבלת “אלוף האלופים” ציבורית עם פרסום שיא אופציונלי לאחר מעבר השלב הראשון.
- בחירת קושי ממוסדת: קל, בינוני, קשה וקשה מאוד, עם טווחי כפל ברורים לכל רמה.
- אפשרות להגבלת זמן: 25 שניות לכל תרגיל.
- מערכת עולמות: כל 25 תשובות נכונות עולים שלב לעולם חדש עם מראה, מבוך ויריבים שונים.
- תרגול אדפטיבי: טעויות בכפל נשמרות ומופיעות שוב מדי פעם לתרגול ממוקד.
- משימות קצרות בזמן המשחק, כמו רצף תשובות, ניצחון על יריבים או איסוף נקודות.
- פידבק לימודי משופר: תשובה נכונה מקבלת חיזוק חיובי, ותשובה שגויה מקבלת חיזוק עם התשובה הנכונה.
- שאלות כפל מדורגות, רצף ניקוד, בונוס חיים כל 25 תשובות נכונות.
- HUD מלא בעברית, מד התקדמות ל-100 תשובות, מסכי התחלה/סיום, ניקוד שיא מקומי.
- ויזואליזציה מחודשת עם קירות נאון, כוכבים, חלקיקים, רעידות, זיקוקים, פולסים ונגיעות סאונד.
- תמיכה במקלדת, נומרית, החלקה במסך ובקרי מגע.
