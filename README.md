# פאקמן מתמטי

משחק עצמאי ונפרד מהאתר הראשי. פותחים את `index.html` בדפדפן או מריצים שרת מקומי מתוך התיקייה:

```bash
python3 -m http.server 5178
```

ואז נכנסים לכתובת:

```text
http://localhost:5178
```

## מה שופר

- מבוך קבוע ויציב במקום מחסומים אקראיים שעלולים לחסום את המשחק.
- תנועה מבוססת זמן, התנגשויות מעוגלות, ותור כיוון כמו במשחקי Pac-Man.
- רוחות עם בחירת מסלול, מצבי רדיפה/פיזור, צבעים ואנימציית עיניים.
- מסך פתיחה שמבקש את שם השחקן לכל משחק ומציג אותו במסך הניצחון.
- שאלות כפל וחילוק מדורגות, רצף ניקוד, בונוס חיים כל 25 תשובות נכונות.
- HUD מלא בעברית, מד התקדמות ל-100 תשובות, מסכי התחלה/סיום, ניקוד שיא מקומי.
- ויזואליזציה מחודשת עם קירות נאון, כוכבים, חלקיקים, רעידות, זיקוקים ונגיעות סאונד.
- תמיכה במקלדת, נומרית, החלקה במסך ובקרי מגע.
